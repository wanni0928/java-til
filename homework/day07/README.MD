# HomeWork#7 Rect
* 아래 테스트 코드가 정상 동작하도록 클래스들을 완성하시오.
* getArea(): 사각형의 넓이를 반환한다.
* getCenterOfMass(): 사각형의 질량중심을 반환한다.
* GetAllPoints(): 사각형의 네 점을 배열로 반환한다.
* rot90(): Pivot을 기준으로 사각형을 90도 회전시킨다.
